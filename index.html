Idee - Projectbeschrijving:

De Onverslaanbare AI - Onze puzzel start met het klassieke spel Blad, Steen, Schaar, maar met een twist: de AI kiest altijd hetzelfde als de speler, waardoor winnen onmogelijk is. De uitdaging? Vind een manier om toch te winnen. De oplossing ligt in een interactieve puzzeldoos. Rond de doos zie je symbolen en bijhorende hendels met LED-indicatoren. Door de juiste hendels in te stellen zodat alle lampjes groen zijn, activeer je een knop die een verborgen luik opent. Daarin vind je een afstandsbediening en ontvang je via een lampje een Morsecode. Deze code gebruik je om via de afstandsbediening cijfers in te voeren op een 7-segment display, dat geactiveerd wordt door een lichtsensor op de doos af te dekken. Voer je de juiste code in, dan krijg je de kans om de AI eindelijk te verslaan in Blad, Steen, Schaar.
